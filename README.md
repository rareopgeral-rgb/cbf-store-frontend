# CBF Store Clone - Entrega

Arquivos principais:

- `index.html`: pagina completa
- `styles.css`: estilos responsivos
- `app.js`: interacoes, carrossel, drawer e checkout
- `assets/`: imagens usadas pela pagina
- `assets/original/`: imagens originais baixadas da URL de referencia

Como visualizar:

1. Abra `index.html` no navegador, ou
2. Rode um servidor local nesta pasta:

```bash
python3 -m http.server 5173
```

Depois acesse:

```text
http://127.0.0.1:5173/
```
